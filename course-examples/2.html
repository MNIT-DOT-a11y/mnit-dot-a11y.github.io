Example 2: Modal Without Focus Management

Activating this button will cause a modal to appear. Keyboard focus should normally be moved to the first button in the modal.

This is a modal window. Try to shift focus away from the modal or to close it while only using the keyboard. The keyboard will cause focus to move outside of the modal. Normally, a modal should trap keyboard focus until it is closed.
