Use of Color

Use of color to convey information: WCAG requires that we not use color alone (or visual characteristics) to convey information. Examples are when a chart uses color alone or when a site uses color-coded text to convey information. This is important for users who have any amount of color deficiency, or color blindness.

Informative Language

Informative text refers to any text or content that conveys critical information, instructions, status, actions, or distinctions—meaning, information that users need to understand or interact with the content.

Examples:

Color Only - Inaccessible With Informative Text - Accessible
Error: fields are in red. Error! Please complete all required fields.
Green button to continue. Select the Continue button to proceed.
YOUR PASSWORD EXPIRES 3 DAYS! Your password EXPIRES in 3 days!

Recap

Use of Color (WCAG 1.4.1):

Recap: Requires that color is not the only visual means of conveying information, indicating an action, or prompting a response.

Best Practices for testing:

  • Verify that any information is not conveyed through color alone. Information can be perceived in other ways.
    • Underlining links in addition to coloring them.
    • Add text, shapes, or patterns to lines and bars in charts.
    • Informational labels next to form fields.
    • Additional informative language expressing the importance.
