When performing magnification and zoom testing, accessibility testers should focus on these key WCAG 2.1 Level AA success criteria when testing zoom and magnification.

1.4.4 Text Resizing

  • Ensure that text can be resized up to 200% without assistive technology and without losing content or functionality.
  • Verify that text-based controls also scale correctly.
  • Confirm that text containers resize when text resizes.
  • Assess the overall readability of the content at different zoom levels.

1.4.10 Reflow

  • Content can reflow when zoomed to 400% without requiring horizontal scrolling.
  • Ensure that the layout adjusts appropriately when zoomed.
  • Verify that all content remains visible and is not cut off when scaled.
  • Check that the page layout does not break or overlap significantly when scaled up.
  • Test that all interactive elements (buttons, links, forms) remain functional when scaled.
  • If provided, test any on-page controls that allow users to increase text size

1.4.12 Text Spacing

  • Verify that no loss of content or functionality occurs when users adjust text spacing properties.
