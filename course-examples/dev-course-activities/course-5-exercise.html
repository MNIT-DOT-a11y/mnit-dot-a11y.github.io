Inclusive CSS Practices Exercise

Hands-on Activity

For this hands-on activity, your task is to enhance an HTML page using accessible CSS techniques. Start with the provided HTML Template below, and apply the principles of color contrast, focus management, and responsive design that you've learned. Focus on making the visual presentation and interactive elements highly usable for everyone, including those relying on assistive technologies.

        /* Basic Reset & Overall Styling */
        body {
            font-family: Arial, sans-serif;
            margin: 0;
            padding: 0;
            background-color: #f0f2f5;
            color: #333;
            line-height: 1.6; /* Accessibility: Improved readability */
        }

        header {
            background-color: #0056b3;
            color: white;
            padding: 20px;
            text-align: center;
        }

        nav ul {
            list-style: none;
            padding: 0;
            margin: 0;
            display: flex;
            justify-content: center;
            gap: 20px;
        }

        nav a {
            /* WCAG Contrast Fix: Changed color to provide sufficient contrast against a potentially light background.
             * Calculated contrast ratio of #0056b3 (header background) with #FFFFFF (new link color) is 7.27:1,
             * which passes WCAG AA (4.5:1) and AAA (7:1) for normal text. */
            color: #ffffff; /* Accessible color for contrast */
            text-decoration: none;
            padding: 8px 15px;
            border-radius: 5px;
            /* Focus Indicator Fix: Re-added 'outline' to the transition for visible focus state. */
            transition: background-color 0.3s ease, outline 0.3s ease;
        }

        /* Focus Indicator Fix: Added a clear visual indicator for both :focus and :hover states.
         * This is crucial for keyboard users. */
        nav a:focus,
        nav a:hover {
            background-color: #004085; /* Darker blue for hover/focus background */
            outline: 3px solid #66ccff; /* Highly visible outline for focus/hover */
            outline-offset: 2px; /* Space between outline and element */
        }


        main {
            max-width: 1200px;

        }

        .action-button:hover {
            background-color: #218838;
        }

        /* Accessibility: Added focus state for buttons */
        .action-button:focus {
            outline: 3px solid #66ccff; /* Clear focus indicator */
            outline-offset: 2px;

            margin-bottom: 20px;
            text-align: left;

            font-size: 1rem;

        .form-group input,
        .form-group textarea {

            padding: 10px;
            border: 1px solid #ccc;
            border-radius: 4px; /* Slightly increased border-radius for softer look */
            box-sizing: border-box; /* Ensures padding/border are included in width */
            /* Focus Indicator Fix: Removed 'outline: none;' and added a custom focus style. */
            outline: none; /* Remove default browser outline */
            transition: border-color 0.3s ease, box-shadow 0.3s ease; /* Smooth transition for focus */
        }

        /* Focus Indicator Fix: Added clear focus styles for form inputs and textareas. */
        .form-group input:focus,
        .form-group textarea:focus {
            border-color: #005fcc; /* Changes border color on focus */
            box-shadow: 0 0 0 3px rgba(0, 95, 204, 0.4); /* Adds a clear, accessible focus indicator */
        }

        /*
         * Mini-Task 1: Typography Refactoring
         */

        html {
            /* Set a base font-size for 'rem' units.
             * Using 100% means 1rem will be equal to the user's default browser font size (typically 16px).
             * This is crucial for accessibility, allowing users to scale text to their preference. */
            font-size: 100%;
        }

        h1 {
            /* Set font-size using 'rem', relative to the html base font-size */
            font-size: 2.5rem; /* e.g., 2.5 * 16px = 40px */
        }

        h2 {
            /* Set font-size using 'rem' */
            font-size: 2rem; /* e.g., 2 * 16px = 32px */
        }

        p {
            /* Set line-height to improve readability. A value of 1.5 to 1.7 is generally recommended. */
            line-height: 1.6;
            /* Set margin-bottom for better spacing between paragraphs. */
            margin-bottom: 1.2rem;
        }

        nav a,
        .action-button,
        .form-group label,
        .form-group input,
        .form-group textarea,
        .read-more { /* Don't forget .read-more links! */
            /* Set font-size using relative units (e.g., 1rem).
             * This ensures all text elements scale consistently with user preferences. */
            font-size: 1rem;
        }

        /*
         * Mini-Task 2: Layout Refactoring (Feature Grid)
         */
        .feature-grid {
            /* Implement CSS Flexbox properties here */
            display: flex;
            /* Allows items to wrap to the next line when space is insufficient. */
            flex-wrap: wrap;
            /* Adds consistent space between flex items. */
            gap: 20px;
            /* Centers items horizontally when there's extra space, improving visual balance. */
            justify-content: center;
        }

        .feature-item {
            border: 1px solid #eee;
            border-radius: 5px;
            padding: 20px;
            text-align: left;
            background-color: #f9f9f9;
            /* Flex shorthand: grow (1), shrink (1), base width (280px).
             * Items will grow to fill available space, shrink if needed (but not below 280px),
             * and have a preferred width of 280px. This creates a responsive grid-like layout. */
            flex: 1 1 280px;
        }

        /* A placeholder class for read-more links that might appear in HTML */
        .read-more {
            display: inline-block; /* Ensures padding/margin work correctly */
            margin-top: 10px;

        }

            outline-offset: 2px;
        }
