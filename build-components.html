Components Library

Alert Component

Learn how to create accessible alert notifications.

<div role="alert" aria-live="assertive">
    <p>Your changes have been saved successfully.</p>
</div>

Ensure the alert is visible and screen-reader accessible. Use aria-live regions to convey urgency appropriately.
