ArcGIS apps mostly rely on standard web keyboard navigation plus a few app‑specific behaviors. Below is a reference you can use when testing published web maps, Dashboards, and StoryMaps.

When doing keyboard testing on ArcGIS maps, Dashboards, and StoryMaps:

  • Navigate the entire experience with keyboard only, ensuring all interactive elements are reachable, operable, and have visible focus.
  • Check that map interactions (panning, zooming, opening pop‑ups) and dashboard interactions (filters, selectors) are fully usable without a mouse.

Core Navigation Keys

Shortcut Action
Tab Move focus forward through all interactive elements (links, buttons, widgets, map, media, etc.)
Shift + Tab Move focus backward through interactive elements.
Enter or Space bar Activate the focused control (equivalent to click; opens panels, activates buttons, toggles checkboxes, etc.).
Arrow keys Move within a focused control (for example, through menu items, tabs, or options) and, in maps, pan the view depending on the app.
Esc Close dialogs and menus or return focus from embedded media or maps to surrounding content.

Published web maps inside apps

When a web map is embedded in an ArcGIS app (such as in Dashboards or StoryMaps), test using the core navigation keys to verify:

  1. The map receives focus via Tab (there is a visible focus ring on the map container or a control).
  2. All visible map tools (zoom, home, basemap switcher, legend, etc.) are reachable and operable via Tab/Enter.

ArcGIS Dashboards keyboard testing

Dashboards support full keyboard navigation through the layout and individual elements. Test dashboard using the core navigation keys to verify:

  1. Focus order is logical as you Tab through the page (top‑to‑bottom, left‑to‑right, grouping related controls together).
  2. Every interactive element shows a visible focus indicator and can be used with Enter/Space.

StoryMaps reading/navigation shortcuts

For readers of classic Story Map Journal/Series and modern ArcGIS StoryMaps, keyboard navigation lets you move through the narrative and media.

Shortcut Action
Tab Move forward through the story header, section navigation, narrative text, and media frames.
Shift + Tab Move backward through the story header, section navigation, narrative text, and media frames.
Enter or Space bar Activate focused buttons such as "Skip to narrative".Send keyboard focus to media, links, and embedded media controls.
Arrow keys Scroll narrative panels and sometimes move between items depending on block type; they can also be used by screen readers to move within content.
EscWhen focus is inside an image or map in classic Story Map Journal, Esc returns focus to the narrative panel
