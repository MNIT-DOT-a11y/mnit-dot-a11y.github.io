Instructions to Install and Run Axe-Core Selenium scripts with Java

This guide provides step-by-step instructions for setting up your environment and running an accessibility scan on a web page using **Java**, **Axe-core**, and **Selenium**.

1. Download and Install Java Development Kit (JDK)

[cite_start]

You need the JDK to compile and run Java applications[cite: 3].

[cite_start]**Download JDK:** Go to the official Oracle JDK download page (or choose an OpenJDK distribution like Adoptium Temurin)[cite: 4]. [cite_start]Choose the Windows installer (.exe) that matches your system (usually x64)[cite: 5].

[cite_start]**Run the Installer:** Double-click the downloaded `.exe` file and follow the on-screen instructions[cite: 6]. [cite_start]The default installation path (e.g., `C:\Program Files\Java\jdk-[version]`) is usually fine[cite: 7].

**Set `JAVA_HOME` Environment Variable:**

    [cite_start]
  1. Right-click the **Start** button and select **System**[cite: 9].
    2. [cite_start]
  2. Click **Advanced system settings**[cite: 10].
    3. [cite_start]
  3. In the **System Properties** window, click **Edit the System Environment Variables for your account**[cite: 11].
    4. [cite_start]
  4. Under **System variables**, click **New...**[cite: 12].
    5. [cite_start]
  5. For **Variable name**, enter `JAVA_HOME`[cite: 13].
    6. [cite_start]
  6. For **Variable value**, enter the path to your JDK installation directory (e.g., `C:\Program Files\Java\jdk-17`)[cite: 14].
    7. [cite_start]
  7. Click **OK**[cite: 15].

**Update the Path Variable:**

    [cite_start]
  1. In the **System variables** section, find the `Path` variable and click **Edit...**[cite: 17].
    2. [cite_start]
  2. Click **New** and add `%JAVA_HOME%\bin`[cite: 18].
    3. [cite_start]
  3. Make sure this entry is moved up to a high priority to ensure the correct JDK is used[cite: 19].
    4. [cite_start]
  4. Click **OK** on all open windows[cite: 20].

[cite_start]**Verify Installation:** Open a new Command Prompt or PowerShell window and type `java -version` and `javac -version`[cite: 21, 23, 24]. [cite_start]You should see the installed JDK version information[cite: 25].

2. Download and Install Apache Maven

[cite_start]

Maven is a build automation tool used for Java projects[cite: 27].

[cite_start]**Download Maven:** Go to the Apache Maven download page[cite: 28]. [cite_start]Download the **Binary zip archive** (e.g., `apache-maven-3.9.6-bin.zip`)[cite: 29].

[cite_start]**Extract Maven:** Unzip the downloaded archive to a location like `C:\Program Files\Apache\apache-maven-[version]`[cite: 30].

[cite_start]**Set `MAVEN_HOME` (or `M2_HOME`) Environment Variable:** Follow the same steps as for `JAVA_HOME`[cite: 32].

    [cite_start]
  1. For **Variable name**, enter `MAVEN_HOME` (or `M2_HOME`)[cite: 33].
    2. [cite_start]
  2. For **Variable value**, enter the path to your Maven installation directory (e.g., `C:\Program Files\Apache\apache-maven-3.9.6`)[cite: 34].
    3. [cite_start]
  3. Click **OK**[cite: 35].

**Update the Path Variable:**

    [cite_start]
  1. In the **System variables** section, find the `Path` variable and click **Edit...**[cite: 37].
    2. [cite_start]
  2. Click **New** and add `%MAVEN_HOME%\bin` (or `%M2_HOME%\bin`)[cite: 38].
    3. [cite_start]
  3. Click **OK** on all open windows[cite: 39].

[cite_start]**Verify Installation:** Open a new Command Prompt or PowerShell window and type `mvn -version`[cite: 40, 42]. [cite_start]You should see the Maven version information[cite: 43].

3. Clone Axe-core Selenium Repository

[cite_start]

You'll download the **axe-core-maven-html** repository using Git[cite: 45].

[cite_start]**Install Git:** If you don't have Git installed, download it from git-scm.com and follow the installation instructions[cite: 46].

[cite_start]**Clone the Repository:** Open your terminal and navigate to the directory where you want to save the repository (e.g., `C:\Users\rubi1dou\Documents`)[cite: 47, 49].

Run the `git clone` command: 

git clone https://github.com/dequelabs/axe-core-maven-html.git
[cite_start]This will create a new folder named `axe-core-maven-html` containing the repository's contents[cite: 53].

4. Create and Configure LiveAxeCoreTest.java

[cite_start]**Create the file:** Inside the cloned repository, navigate to `axe-core-maven-html/selenium/src/test/java/com/yourcompany/axtest/`[cite: 56]. [cite_start]Create a new file named `LiveAxeCoreTest.java` (if the directories don't exist, create them)[cite: 57].

[cite_start]**Add the code:** Copy and paste the provided JUnit 4 code into the new file[cite: 58]. [cite_start]This code uses the `Parameterized` runner to test multiple sites[cite: 59].

        System.out.println("Navigating to page: " + siteUrl);
        try {
            driver.get(siteUrl);
            AxeBuilder axeBuilder = new AxeBuilder();
            System.out.println("Executing axe-core accessibility scan on " + siteUrl + "...");
            Results axeResults = axeBuilder.analyze(driver);
            System.out.println("Axe-core scan completed for " + siteUrl + ".");

            ObjectMapper mapper = new ObjectMapper();
            File jsonReportDir = new File("target/a11y-json-reports");
            if (!jsonReportDir.exists()) {
                jsonReportDir.mkdirs();
            }
            String jsonFileName = siteUrl.replaceAll("[^a-zA-Z0-9.-]", "_") + "_axe_results.json";
            File jsonFile = new File(jsonReportDir, jsonFileName);
            mapper.writerWithDefaultPrettyPrinter().writeValue(jsonFile, axeResults);
            System.out.println("Axe-core JSON results saved to: " + jsonFile.getAbsolutePath());

            generateHtmlReport(axeResults, siteUrl);

            if (axeResults.getViolations().isEmpty()) {
                System.out.println("No accessibility violations found on " + siteUrl + "!");
                System.out.println("This means the page appears to be highly accessible.");
            } else {
                System.out.println("--- Accessibility Violations Found on " + siteUrl + " ---");
                System.out.println("Review these issues to improve the accessibility of the page.");
                axeResults.getViolations().forEach(violation -> {
                    System.out.println("Rule ID: " + violation.getId());
                    System.out.println("Description: " + violation.getDescription());
                    System.out.println("Help URL: " + violation.getHelpUrl());
                    System.out.println("Impact: " + violation.getImpact());
                    System.out.println("Nodes:");
                    violation.getNodes().forEach(node -> {
                        System.out.println("   HTML: " + node.getHtml());
                        System.out.println("   Target: " + node.getTarget());
                        System.out.println("   Failure Summary: " + node.getHtml());
                        System.out.println("-----");
                    });
                    System.out.println("------------------------------------");
                });
            }

            Assert.assertTrue("Accessibility violations found on " + siteUrl + ": " + axeResults.getViolations().toString(),
                axeResults.getViolations().isEmpty());
        } catch (AxeRuntimeException e) {
            System.err.println("Axe-core runtime error during analysis of " + siteUrl + ": " + e.getMessage());
            e.printStackTrace();
        } catch (IOException e) {
            System.err.println("Error saving Axe-core results to JSON or HTML for " + siteUrl + ": " + e.getMessage());
            e.printStackTrace();
        } catch (Exception e) {
            System.err.println("An unexpected error occurred while testing " + siteUrl + ": " + e.getMessage());
            e.printStackTrace();
        }
    }
