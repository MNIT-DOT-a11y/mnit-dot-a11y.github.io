Axe-Core Cypress Setup (for C# Teams)

This guide explains how to run axe-core accessibility scans using Cypress. While Cypress tests are written in JavaScript, they are often used by C# teams in CI/CD workflows.

1. Install Node.js

  • Download Node.js LTS from nodejs.org
  • Verify with node --version and npm --version

2. Create a Cypress Project


mkdir axe-cypress-tests
cd axe-cypress-tests
npm init -y
npm install cypress --save-dev

3. Add Axe-Core


npm install cypress-axe axe-core --save-dev

4. Configure Cypress

Edit cypress/support/e2e.js and add:

import 'cypress-axe';

5. Create Accessibility Test


describe('Accessibility checks', () => {
  it('Runs axe-core on the page', () => {
    cy.visit('https://www.example.com');
    cy.injectAxe();
    cy.checkA11y();
  });
});

6. Run Tests

  • npx cypress open for interactive runner
  • npx cypress run for headless CI mode

7. C# Integration

  • Run Cypress alongside .NET builds
  • Fail builds on accessibility violations
  • Parse Cypress JSON output for C# dashboards
