Instructions to Install and Run Axe-Core Cypress Tests (for C# Teams)

This guide explains how to run axe-core accessibility scans using Cypress. Although Cypress tests are written in JavaScript, they are commonly used by C# development teams as part of CI/CD pipelines and shared accessibility testing strategies.

1. Install Node.js

  • Download Node.js from the official website
  • Install the LTS version
  • Verify installation using node --version

2. Create a Cypress Project


mkdir axe-cypress-tests
cd axe-cypress-tests
npm init -y
npm install cypress --save-dev

3. Add Axe-Core for Cypress


npm install cypress-axe axe-core --save-dev

4. Configure Cypress

Add axe-core support to Cypress:


import 'cypress-axe';

5. Create an Accessibility Test


describe('Accessibility checks', () => {
  it('Runs axe-core on the page', () => {
    cy.visit('https://www.example.com');
    cy.injectAxe();
    cy.checkA11y();
  });
});

6. Run the Tests

  • Open Cypress UI: npx cypress open
  • Run headless: npx cypress run
  • Review accessibility violations in test output

7. Using Cypress Results in C# Workflows

  • Run Cypress alongside .NET builds in CI
  • Fail builds when accessibility violations occur
  • Parse Cypress output for reporting dashboards
  • Use Cypress for UI testing and C# for backend validation
