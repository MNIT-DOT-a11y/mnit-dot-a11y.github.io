Select Component

A select component, also known as a dropdown menu, allows users to choose one option from a list. It is a fundamental form element used to save space when there are many options that might otherwise take up too much vertical space.

Passed WCAG 2.1 AA

Interactive Example

Standard Select Menu

Source Code

<div class="select-wrapper">
    <label for="country-select">Choose a country:</label>
    <select id="country-select">
        <option value="" disabled selected>-- Select an option --</option>
        <option value="us">United States</option>
        <option value="ca">Canada</option>
        <option value="mx">Mexico</option>
    </select>
</div>

Guidance

Select menus are best for lists with more than five to seven options. For shorter lists, consider using radio buttons to make all options visible at once, reducing the number of clicks required.

When to use the select component

  • Forms with many options. When a form requires users to choose from a long list, like a list of states or countries.
  • Filtering. To filter content on a page based on a single selection from a long list of categories.
