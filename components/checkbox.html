Checkbox Component

This guide provides instructions for creating a checkbox component that meets accessibility standards.

Interactive Example

Source Code

<div class="checkbox-container">
    <label>
        <input type="checkbox" id="subscribeCheckbox" name="subscribe" />
        Subscribe to newsletter
    </label>
    <label>
        <input type="checkbox" id="termsCheckbox" name="agreeTerms" />
        Agree to terms and conditions
    </label>
</div>

Accessibility Features

  • Labels are associated with checkboxes to enhance accessibility for screen readers.
  • Check that the checkboxes are operable using keyboard navigation.
  • Each input has a unique id attribute that corresponds to its label.
