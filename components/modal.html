Pagination Component

Pagination allows users to navigate through a large set of content by dividing it into multiple pages. This component helps to prevent information overload and improves performance on pages with many items.

Source Code

<nav aria-label="Pagination">
  <ul class="pagination">
    <li class="pagination-item">
      <a href="#" aria-label="Previous page">Previous</a>
    </li>
    <li class="pagination-item"><a href="#">1</a></li>
    <li class="pagination-item active"><a href="#" aria-current="page">2</a></li>
    <li class="pagination-item"><a href="#">3</a></li>
    <li class="pagination-item">
      <a href="#" aria-label="Next page">Next</a>
    </li>
  </ul>
</nav>

Accessibility Guidance

  • Use a `nav` element with a descriptive `aria-label` to identify the pagination section.
  • Use `aria-label` on "Previous" and "Next" links to provide more context to screen reader users.
  • The current page should be marked with `aria-current="page"`.
  • Use `role="list"` and `role="listitem"` if not using `ul` and `li` tags.
