Typography

Accessible typography ensures that text is readable and legible for all users. This includes proper heading structure, font choices, line height, and color contrast.

Interactive Example

Heading 1 (H1)

This is a paragraph of standard body text. It should have sufficient line-height for readability and strong color contrast against the background.

Heading 2 (H2)

This is another paragraph. Headings should be used hierarchically to outline the content structure of the page.

Heading 3 (H3)

This paragraph contains an unordered list:

  • List item 1
  • List item 2
  • List item 3

Heading 4 (H4)

This paragraph contains an ordered list:

  1. First step
  2. Second step
  3. Third step

<h1>Heading 1 (H1)</h1>
<p>This is a paragraph of standard body text. It should have sufficient line-height for readability and strong color contrast against the background.</p>

<h2>Heading 2 (H2)</h2>
<p>This is another paragraph. Headings should be used hierarchically to outline the content structure of the page.</p>

<h3>Heading 3 (H3)</h3>
<p>This paragraph contains an unordered list:</p>
<ul>
    <li>List item 1</li>
    <li>List item 2</li>
    <li>List item 3</li>
</ul>

<h4>Heading 4 (H4)</h4>
<p>This paragraph contains an ordered list:</p>
<ol>
    <li>First step</li>
    <li>Second step</li>
    <li>Third step</li>
</ol>

Accessibility Guidance

  • Use a logical heading structure (`

    ` down to `

    `) without skipping levels.
  • Ensure font size is large enough to be easily readable, with a minimum of `16px` for body text.
  • Use sufficient color contrast between text and background, at least a ratio of 4.5:1.
  • Use clear, simple fonts and avoid styling that makes text difficult to read (e.g., all caps, excessive italicization).
