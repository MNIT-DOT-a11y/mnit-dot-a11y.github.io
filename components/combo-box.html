Combo Box Component

This guide provides instructions for creating a combo box component that meets accessibility standards.

Interactive Example

  • Option 1
  • Option 2
  • Option 3

Source Code

<div class="combo-box-container">

    <label for="combo-box" class="combo-box-label">Choose an option:</label>

    <input type="text" id="combo-box" class="combo-box-input" aria-haspopup="listbox" aria-expanded="false" aria-controls="combo-box-list" aria-autocomplete="list" role="combobox" placeholder="Select an option..." />

    <ul id="combo-box-list" class="combo-box-dropdown" role="listbox">

        <li class="combo-box-dropdown-item" role="option" tabindex="0">Option 1</li>

        <li class="combo-box-dropdown-item" role="option" tabindex="0">Option 2</li>

        <li class="combo-box-dropdown-item" role="option" tabindex="0">Option 3</li>

    </ul>

</div>

Accessibility Features

  • The combo box uses role="combobox", aria-haspopup, and aria-expanded for accessibility.
  • Dropdown options are wrapped in a listbox with individual option roles.
  • Ensure the combo box is operable using keyboard navigation.

Testing

To ensure the combo box component meets accessibility standards, refer to the Combo Box Component Testing Checklist.
