Breadcrumb Component

This guide provides instructions for creating a breadcrumb component that meets accessibility standards.

Interactive Example

  1. Home
  2. Library
  3. Data

Source Code

<nav aria-label="breadcrumb">
    <ol class="breadcrumb">
        <li class="breadcrumb-item"><a href="#">Home</a></li>
        <li class="breadcrumb-item"><a href="#">Library</a></li>
        <li class="breadcrumb-item active" aria-current="page">Data</li>
    </ol>
</nav>

Guidance

When using breadcrumb components, it is important to provide users with a clear and easy way to navigate back to previous sections of the site.

  • Use breadcrumbs for hierarchical navigation.
  • Ensure links are distinct and clearly formatted.
  • Use aria-current="page" to indicate the current page for accessibility.
  • Keep the breadcrumb path consistent with the site structure.

Accessibility Features

  • The breadcrumb uses aria-label="breadcrumb" to provide a label for assistive technologies.
  • Each item is clearly defined, and the current page is marked with aria-current="page" to indicate the user's position.
