Tag Component

A tag is a small label used to categorize or highlight a key attribute of an item. Tags are often used to filter content, display status, or provide a quick summary of a topic. An accessible tag component ensures its purpose is clear to all users, and if interactive, it is easily operable via keyboard and screen reader.

Passed WCAG 2.1 AA

Source Code

<!-- A simple tag list for categories -->
<div class="tag-list">
    <span class="tag">Category 1</span>
    <span class="tag">Category 2</span>
    <span class="tag">Category 3</span>
</div>

<!-- A list of interactive tags for filters -->
<div class="tag-list">
    <button class="tag-interactive" aria-label="Remove Filter: Blue">
        Blue <i class="fa-solid fa-xmark tag-close-icon" aria-hidden="true"></i>
    </button>
    <button class="tag-interactive" aria-label="Remove Filter: Red">
        Red <i class="fa-solid fa-xmark tag-close-icon" aria-hidden="true"></i>
    </button>
    <button class="tag-interactive" aria-label="Remove Filter: Green">
        Green <i class="fa-solid fa-xmark tag-close-icon" aria-hidden="true"></i>
    </button>
</div>

Guidance

Tags are an effective way to visually organize information on a page. The key is to use them appropriately: if a tag is clickable and performs an action, it should be a button. If it's simply a piece of descriptive text, a `` is the correct choice.

When to use the tag component

  • Content categorization. To label blog posts, products, or other items with relevant keywords.
  • Filters and selections. To display filter criteria that can be removed by the user, providing a clear visual cue.

Usability tips

  • Clear purpose. If a tag is interactive, its purpose should be clear from its label and visual style.
  • Consistent design. All tags should have a consistent look and feel, making them easy to recognize.

Accessibility Features

  • **Semantic HTML.** Using `
  • **Accessible Names.** Interactive tags with an icon should have a clear `aria-label` that describes the button's action, such as "Remove Filter: Blue".
  • **Decorative Icons.** The icon within the interactive tag is decorative, so `aria-hidden="true"` is used to prevent screen readers from announcing it redundantly.
