Tag

Tags are used to categorize or label content. They can be interactive (e.g., clickable filters) or static labels. Proper accessibility ensures their function is clear to all users.

Interactive Example

Category 1
Item 2
Category 3

Source Code

<div class="tag-container">
    <a href="#" class="tag interactive" role="button">Category 1</a>
    <div class="tag" role="status">
        Item 2
        <button class="close-btn" aria-label="Remove Item 2">
            <i class="fa-solid fa-xmark" role="img" aria-label="Remove" aria-hidden="true"></i>
        </button>
    </div>
    <a href="#" class="tag interactive" role="button">Category 3</a>
</div>
