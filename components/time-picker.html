Time Picker

A time picker allows users to select a time. The most accessible solution is to use the native HTML5 `` as it is well-supported by browsers and assistive technologies.

Interactive Example

Source Code

<div class="form-group">
    <label for="event-time">Event Time:</label>
    <input type="time" id="event-time" name="event-time">
</div>

Accessibility Guidance

  • Use <input type="time"> as the primary choice for time input.
  • Always associate the input with a `
  • Avoid custom-built time pickers unless they are rigorously tested to meet accessibility standards for keyboard navigation, focus management, and screen reader support.

Testing

To ensure the component meets accessibility standards, refer to the Time Picker Testing Checklist.
