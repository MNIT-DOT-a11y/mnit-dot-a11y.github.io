Tooltip Component

A tooltip is a small, contextual popup that appears on hover or focus, providing additional information or context for a user interface element. An accessible tooltip is designed to be triggered by both mouse and keyboard interactions and has a clear way to be dismissed.

Passed WCAG 2.1 AA

Interactive Example

Standard Tooltip

This is the tooltip content. It provides extra information.

Source Code

<div class="tooltip-container">
    <button class="tooltip-trigger" aria-describedby="tooltip-content">
        Hover or Focus on Me
    </button>
    <span id="tooltip-content" role="tooltip" class="tooltip-popup">
        This is the tooltip content.
    </span>
</div>

Guidance

Tooltips are useful for providing short, helpful pieces of information without taking up permanent space on the page. They should be used to supplement existing content, not to replace it.

When to use the tooltip component

  • Informational icons. Use a tooltip to provide a text label for an icon that might not be immediately clear.
  • Clarifying terms. For providing a definition or explanation of a technical term or concept.

Usability tips

  • Clear trigger. The tooltip should appear on both mouse hover and keyboard focus to be accessible to all users.
  • Simple content. The tooltip's content should be short and to the point. Long explanations should be provided in a separate location.

Accessibility Features

  • **ARIA Attributes.** The `role="tooltip"` attribute identifies the popup content for screen readers. `aria-describedby` links the trigger element to the tooltip content, ensuring the description is announced.
  • **Keyboard Operability.** The tooltip should be dismissible by pressing the Escape key, which is a standard pattern for accessible popups.
  • **Visible on Focus.** The tooltip should be visible when the user focuses on the trigger element via keyboard navigation, not just on mouse hover.

Testing

To ensure the tooltip component meets accessibility standards, refer to the Tooltip Component Testing Checklist.
