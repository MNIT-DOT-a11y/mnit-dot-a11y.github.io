Tooltip

A tooltip provides supplementary information when a user hovers over or focuses on a specific element. For accessibility, tooltips must be keyboard accessible and be announced by screen readers.

Interactive Example

Hover or focus on the following link for more information:

This is the tooltip content. It provides extra information about the link.

Source Code

<div class="tooltip-container">
    <button class="tooltip-trigger" aria-describedby="my-tooltip">Learn More</button>
    <span id="my-tooltip" role="tooltip" class="tooltip-text">
        This is the tooltip content. It provides extra information about the link.
    </span>
</div>

Accessibility Guidance

  • The element that triggers the tooltip (the "trigger") should be focusable, such as a `
  • Use `aria-describedby` on the trigger element to link it to the tooltip's content.
  • The tooltip content should have `role="tooltip"`.
  • Ensure the tooltip appears on both hover (for mouse users) and focus (for keyboard users).
