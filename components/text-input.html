Time Picker Component

A time picker is a graphical user interface element that allows users to select a specific time. This component simplifies the process of entering time values and helps ensure the data is in a consistent and valid format.

Passed WCAG 2.1 AA

Interactive Example

Standard Time Picker

Source Code

<div class="time-picker-wrapper">
    <label for="time-input">Select a time:</label>
    <input type="text" id="time-input" class="time-picker-input" placeholder="HH:MM AM/PM">
</div>

Guidance

Time pickers are an effective way to handle time entry in forms, providing an intuitive interface that reduces user error. The component should be designed to be easily operable with both a mouse and a keyboard.

When to use the time picker component

  • Form fields. When a form requires a specific time input, such as for scheduling appointments or setting reminders.
  • Time-based filtering. For filtering search results or data based on a time of day.

Usability tips

  • Clear labels. The input field must have a visible and correctly associated label to explain its purpose.
  • Keyboard support. The component should be fully navigable and operable using a keyboard, including arrow keys for changing the time.

Accessibility Features

  • Descriptive Labels. The label element is correctly associated with the , providing a clear accessible name for screen readers.
  • Semantic HTML The input element is used correctly within the form. For a rich interactive time picker, additional ARIA attributes (like `aria-expanded`) might be needed to communicate the state of the component.
  • Keyboard Operability. The time picker should allow users to enter time values directly or use keyboard shortcuts to interact with a graphical interface.
