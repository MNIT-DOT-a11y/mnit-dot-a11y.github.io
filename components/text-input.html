Text Input

Accessible text inputs are the foundation of any web form. They must have clear labels, keyboard support, and provide feedback for validation errors.

Interactive Example

Enter your desired username.

Source Code

<form>
    <div class="form-group">
        <label for="username">Username</label>
        <input type="text" id="username" name="username" aria-describedby="username-desc">
        <div id="username-desc" class="description">Enter your desired username.</div>
    </div>
    <div class="form-group">
        <label for="email">Email</label>
        <input type="email" id="email" name="email">
    </div>
</form>

Accessibility Guidance

  • Use a `
  • Use `aria-describedby` to link additional instructional or descriptive text to the input field.
  • Use appropriate `type` attributes (e.g., `email`, `tel`) to provide semantic meaning and help with mobile keyboards.
  • Ensure the input field is keyboard focusable and provides a clear visual focus indicator.
